# Welcome to {{hackweek}}!

![banner](img/banner.png)

The title of this year's event is “Developers’ hackweek: Streamlining and 
expanding ICESat-2 tooling and workflows across libraries and missions”.
 
This hackweek will be be open to developers and advanced users via an 
invitation-only process.

The focus will be on streamlining (via hacking/project work) the current 
ecosystem of ICESat-2 tooling and creating a vision (and maybe some prototypes!)
for how that ecosystem can evolve and expand to include more missions and tools
(especially NISAR, but also GEDI, SnowEx, etc.).

Ideally this work will enable us to make recommendations to NASA on software
unification.

We anticipate few to no formal tutorials with the schedule including focused 
discussions and abundant hacking time.
